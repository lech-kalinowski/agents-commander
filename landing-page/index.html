The Problem

AI agents are powerful. But isolated.

Every agent runs in its own silo. There is no standard way to orchestrate them together from the terminal.

☓

No Standard Protocol

There is no universal way for terminal-based AI agents to communicate. Each tool invents its own integration layer, creating fragmentation.

⛔

Isolated Processes

Claude cannot talk to Codex. Gemini cannot hand off to Aider. Each agent operates in a vacuum, blind to what other agents are doing.

⚠

API-Only Solutions

Existing multi-agent frameworks require API keys, custom code, and cloud infrastructure. Nothing works natively in the terminal where developers live.

The Solution

One terminal. Multiple agents. Zero integration.

The Commander Protocol is a simple, text-based messaging format that any agent can produce and any terminal can route.

commander-protocol.md 
// SEND — direct message to an agent
===COMMANDER:SEND:codex:2===
Write unit tests for src/auth/
===COMMANDER:END===

// REPLY — respond to last sender
===COMMANDER:REPLY===
12 tests passing, 0 failing.
===COMMANDER:END===

// BROADCAST • STATUS • QUERY
// 5 commands, fully bidirectional
// Sender gets ACK after delivery
1

Inject — Skill Files

Agents Commander injects a skill file into each agent's context, teaching it the Commander Protocol message format. No API integration needed.

2

Scan — Output Monitoring

The Commander continuously scans each agent's terminal output for protocol markers. Pure text matching — works with any agent that can print text.

3

Route — Message Delivery

When a marker is detected, Commander extracts the message and injects it into the target agent's stdin. The receiving agent sees it as a new prompt.

Protocol Message Format
===COMMANDER:SEND:{target_agent}:{panel_number}===
{message content — any text, any length}
===COMMANDER:END===

Fields:   SEND — action  |  codex — target agent  |  2 — target panel number

Get Started

Up and running in 30 seconds

One command to install. One command to start collaborating.

terminal 
# Install globally
$ npm install -g agents-commander

# Launch in your project directory
$ cd your-project/
$ agents-commander

# Or run directly with npx
$ npx agents-commander
quick start 
# Press F2 to select an agent (Claude, Codex, Gemini...)
# Press Ctrl+3 for 3-panel layout
# Press Ctrl+B to browse 120 prompt templates
# Agents auto-discover Commander Protocol via skill files

Requires Node.js 18+  •  Works on macOS, Linux, and WSL  •  Full documentation on GitHub
